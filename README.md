📦 API REST com Firebase

API REST desenvolvida em Node.js utilizando Express e Firebase Firestore para gerenciamento de itens.
O projeto implementa operações CRUD (Create, Read, Update, Delete) e utiliza uma estrutura organizada em Controllers, Models, Routes e Middlewares.

O objetivo do projeto é demonstrar a criação de uma API backend integrada a um banco de dados NoSQL em nuvem, utilizando boas práticas de organização de código.

🚀 Tecnologias Utilizadas

Node.js

Express.js

Firebase Admin SDK

Firebase Firestore

JavaScript

REST API

Middleware de autenticação

⚙️ Funcionalidades

📌 Criar itens no banco de dados

📌 Listar itens cadastrados

📌 Atualizar itens existentes

📌 Remover itens do sistema

📌 Integração com Firebase Firestore

📌 Middleware de autenticação para proteção de rotas

📌 Estrutura organizada em arquitetura modular
